默认样式

div.block{
    position: absolute;
    width: 10px;
    height: 10px;
    background-color: #555;
    -webkit-transform: rotate(90deg);/* 旋转90° */
    transform: rotate(90deg);/* 旋转90° */
    animation-iteration-count: infinite;/* 动画执行次数-无限循环 */
    animation-timing-function: ease-out;/* 动画的速度曲线-动画以低速结束 */
    animation-duration: 2s;/* 动画执行周期-2s */
    animation-play-state: paused; 
}

1.左上角

.block-1 {
    left: calc(200px - 10px * 1.5);
    top: calc(200px - 10px * 1.5);
    animation: block1 1s;
}
@keyframes block1 {
    /* 初始状态：放大3倍 从左上角到中间 */
    from {
        left: calc(-10px * 1.5);
        top: calc(-10px * 1.5);
        transform: scale(3);
    }
}
